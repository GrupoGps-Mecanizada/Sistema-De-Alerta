🔗 Integração GitHub

ℹ️ Como criar um Personal Access Token:
  1. Acesse GitHub → Settings → Developer settings → Personal access tokens
  2. Clique em "Generate new token" → "Generate new token (classic)"
  3. Nome: "Sistema Alertas GrupoGPS"
  4. Permissões necessárias: repo, read:user, workflow
  5. Copie o token e cole no campo acima
⚠️ Para desabilitar notificações de GitHub Actions: Use o botão "Desabilitar Actions" ou acesse Settings → Actions → Disable Actions no repositório
